This program is a piano tiles bot
The game can be found on this website http://tanksw.com/piano-tiles/
I used this video https://www.youtube.com/watch?v=YRAIUA-Oc1Y&t=358s
as a reference for making this

This was made using

	*keyboard
	*win32api
	*win32con
	*threading
